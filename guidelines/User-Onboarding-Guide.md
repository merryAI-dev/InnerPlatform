# InnerPlatform 사용자 온보딩 가이드

## 시작하기

InnerPlatform은 MYSC의 사업관리 통합 플랫폼입니다.
기존 시트 방식의 사업비 관리를 디지털화하여 교차검증, 자산화, 표준화를 지원합니다.

## 역할별 접속

| 역할 | URL | 할 수 있는 것 |
|------|-----|-------------|
| **관리자 (Admin)** | `/` | 전체 프로젝트 관리, 사용자 관리, 감사 로그, 설정 |
| **PM** | `/portal` | 사업비 입력(주간), 예산 관리, 증빙 업로드, 연차 신청 |
| **뷰어 (Viewer)** | `/portal` | 프로젝트 조회, 증빙 드라이브 접근 (읽기 전용) |

## PM 주간 업무 흐름

```
예산 총괄 → 통장내역 정리 → 사업비 입력(주간) → 캐시플로(주간) 비교 → 주차 마감
```

### 1. 사업비 입력(주간)

경로: 포털 → **사업비 입력(주간)**

- 거래 내역을 행 단위로 입력합니다
- **Tab** 키로 다음 셀 이동, **Enter**로 다음 행 이동
- **Ctrl+C/V**: 범위 복사/붙여넣기 (Excel/Google Sheets에서도 가능)
- **Ctrl+X**: 선택 범위 잘라내기
- **Ctrl+Z**: 실행 취소
- 입력 후 **저장** 버튼 → 매주 금요일 6시까지 PM이 제출

### 2. 증빙 관리

- 거래별 **증빙 폴더 생성** → Google Drive에 자동 연결
- 파일 업로드 후 **동기화** → 증빙 자동 집계

### 3. 주차 마감

- 결산완료 후에도 **Projection**은 수정 가능
- 캐시플로 탭에서 계획 vs 실제 비교

## 관리자 주요 기능

### 프로젝트 관리
경로: `/projects`
- 프로젝트 목록 → 상세 → 편집
- 사업 등록 위저드 (계약서 업로드 포함)

### 감사 로그
경로: `/audit`
- 모든 데이터 변경 이력 조회
- CSV 내보내기 가능

### 설정
경로: `/settings`
- 데이터 마이그레이션
- 시스템 설정

## 자주 묻는 질문

**Q: 결산완료 후 수정이 안 돼요**
A: 결산완료 후에도 Projection 수정은 가능합니다. Actual(실제값) 입력만 잠깁니다.

**Q: Excel에서 복사한 데이터가 깨져요**
A: Ctrl+V로 붙여넣기 하면 자동으로 탭/줄바꿈을 인식합니다. 한 셀만 복사하면 선택 범위 전체에 채워집니다.

**Q: 증빙 자동 집계가 안 돼요**
A: 증빙 폴더 생성 후 → 파일 업로드 → **동기화** 버튼을 눌러야 목록이 반영됩니다.
